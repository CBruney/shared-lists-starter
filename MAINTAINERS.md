# Maintainers

## Current Maintainer

- Craig Bruney, GitHub: `@CBruney`

## Responsibilities

Maintainers are responsible for:

- Reviewing security-sensitive changes.
- Keeping auth and list-permission tests current.
- Reviewing database migrations.
- Publishing releases.
- Keeping public docs free of private deployment details.
- Triage for issues, pull requests, and discussions.

## Private Reports

Use GitHub private vulnerability reporting for confidential security reports. For conduct reports that include private personal information, use the same private GitHub reporting route and prefix the report title with `Conduct report`.
