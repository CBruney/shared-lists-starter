# Support

This starter is provided as open-source infrastructure. There is no guaranteed support channel.

Good places to start:

- Read `README.md`.
- Read the hosting doc for your lane.
- Run `npm run check`.
- Check existing issues.
- Ask a setup question in GitHub Discussions.

When opening an issue, include:

- Hosting lane: OpenAI Sites or Cloudflare.
- Node version.
- Browser and device.
- Exact command or URL.
- Expected behavior.
- Actual behavior.
- Any relevant logs with secrets removed.

For confidential security reports, use GitHub private vulnerability reporting instead of public issues or discussions.
