# Code Of Conduct

Be direct, kind, and useful.

Shared Lists Starter should be a practical place for people to learn, fork, and improve a small shared-list app. Harassment, personal attacks, threats, stalking, sustained disruption, and bad-faith behavior are not welcome.

## Expected Behavior

- Assume good faith where it is reasonable.
- Keep criticism specific to the work.
- Respect privacy. Do not post private emails, private URLs, tokens, logs, database exports, or personal information.
- Accept maintainer decisions and move on when a thread is closed.

## Reporting

For ordinary moderation concerns, open a GitHub issue or discussion and keep the report factual.

For a conduct report that includes private personal information, use GitHub private vulnerability reporting and prefix the title with `Conduct report`:

```text
https://github.com/CBruney/shared-lists-starter/security/advisories/new
```

Do not put sensitive personal information in a public issue.

## Enforcement

Maintainers may edit or remove comments, close discussions, block users, or restrict participation when behavior makes the project less safe or useful for others.
